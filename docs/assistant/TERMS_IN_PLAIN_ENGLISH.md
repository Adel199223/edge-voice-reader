# Terms In Plain English

- `Edge Voice Reader`: the lightweight unpacked Edge extension in this repo.
- `background service worker`: the part of the extension that keeps reading even if the popup closes.
- `hidden voice names`: the exact Microsoft Edge voice names for Ava and Andrew that can be spoken directly even when normal voice lists do not show them.
- `read-selection`: the keyboard shortcut that reads the text you selected on the current page.
- `activeTab`: the browser permission that lets the extension read the current page only in the right live tab context.
- `Windows-host validation`: checking the extension in real Microsoft Edge on Windows, not only through WSL commands.
